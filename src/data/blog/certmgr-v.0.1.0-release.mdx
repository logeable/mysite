## 特性
	
- 🏗️ 多命名空间管理，实现证隔离
- 📜 证书全生命周期管理（根证书、中间 CA、终端证书）
- 🔑证书链自动构建与可视化展示
- 🔄 支持证书签发、续签、导出等基础操作
- 🖥️ 基于 Electron + React + Golang 的跨平台桌面应用（目前只测试了Mac环境）

## 主要界面说明

- **左侧菜单**：空间管理、证书管理两大主入口。
- **空间管理**：表格/卡片展示空间列表，支持新建、编辑、删除空间。

  ![空间管理](src/assets/blog-images/certmgr-v.0.1.0-release/namespace.png)

- **证书管理**：空间下拉选择，树状展示证书链，右键菜单支持签发、续签、删除、详情、私钥查看等操作。

  ![证书链可视化](src/assets/blog-images/certmgr-v.0.1.0-release/certificate_tree.png)

- **证书操作**：签发、续签、导出等操作界面。

  ![证书操作](src/assets/blog-images/certmgr-v.0.1.0-release/certificate_op.png)

- **证书详情**：展示证书详细信息。

  ![证书详情](src/assets/blog-images/certmgr-v.0.1.0-release/certificate_detail.png)
